Client documentation

Quick guide to install, configure, and use the Marte client.

1. Installation

Install the client and verify the main command is available in the terminal. 

marte --help

2. Create an environment

Each environment contains the backend configuration and project exploits. 

marte init

3. Configure the backend

Set endpoint and credentials for the active environment.

marte host <ip> <port>
marte login <username> <password>
marte pull
